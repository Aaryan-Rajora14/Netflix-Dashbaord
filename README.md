## 🎯 Project Objective

This project is designed to help students learn how to build a professional, interactive dashboard in Power BI using real-world Netflix data.

## 📊 What This Dashboard Shows

- Top movies & TV shows by rating and popularity
- Content distribution by genre, country, and release year
- Trends in Netflix content addition over time
- Rating analysis and viewer preferences
- Comparison between Movies vs TV Shows
- Key performance insights using KPIs and charts

## 🧑‍🏫 Learning Objectives (For Students)

By working on this project, you will learn:

1. **Data Import & Cleaning** using Power Query
2. **DAX Calculations** – Creating measures for ratings, counts, and trends
3. **Interactive Visualizations** – Bar charts, line charts, pie/donut charts, cards, and slicers
4. **Dashboard Design Principles** – Layout, color theory, and user experience
5. **Storytelling with Data** – Turning raw data into meaningful insights
6. **Professional Theming** using custom JSON theme (CY24SU10)

## 🛠️ Technologies Used

- Power BI Desktop
- Power Query (for data transformation)
- DAX (Data Analysis Expressions)
- Custom Theme (CY24SU10)

## 📁 Project Files

- `Netflix Dashboard.pbix` ← Main Power BI file (ready to open)
- Embedded Netflix logo and assets
- Pre-built DAX queries

## 🚀 How to Use This Project (Step-by-Step for Students)

1. Download the `Netflix Dashboard.pbix` file
2. Open it in **Power BI Desktop** (Free version is sufficient)
3. Explore each page and play with the slicers/filters
4. Go to **"Model"** view to understand relationships
5. Check **"Data"** view to see the cleaned dataset
6. Review the DAX measures in the **DAXQueries** folder
7. Try modifying visuals or creating your own new page

## 📈 Dashboard Pages Included

- **Overview** – Key metrics and overall performance
- **Content Analysis** – Movies vs TV Shows, Genres
- **Trends** – Content addition over the years
- **Ratings & Popularity** – Top titles and rating distribution
- **Country Insights** – Content production by country

## 💡 Tips for Students

- Try changing the filters and observe how visuals interact
- Duplicate a page and experiment with different chart types
- Create your own measure using DAX (example: Average Rating by Genre)
- Customize colors to match your own style
